1. 工具说明
   本工具无侵入,不需要root，低延迟，可以测试游戏，视频，普通应用的性能数据，数据精准，目前支持数据录制，可测试android和ios两个端，小程序暂时不支持。ios17及以上暂时不支持
 后续会不断的优化，包括测量的指标，鸿蒙系统的支持，iOS更高的支持，另会增加ANR的监控
   
   

   提示 ：工具包需要放在不含中文目录里使用（目录里不要有中文）
